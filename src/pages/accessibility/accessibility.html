Accessibility Statement

HardwareLizard is committed to providing a website that is accessible to the widest possible audience. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.2, Level AA.

This statement applies to the HardwareLizard webshop and website at https://1davidungor.github.io/HardwareLizard-webshop/.

Conformance status

The website is designed to be conformant with WCAG 2.2 Level AA. We continuously improve accessibility as the project evolves.

Accessibility features

  • Semantic structure with landmarks (header, navigation, main content, footer).
  • Keyboard operability and visible focus indicators.
  • A “Skip to content” link for faster keyboard navigation.
  • Form inputs with associated labels and required-field validation.
  • Clear status messaging for plan selection and checkout steps.

Testing and evaluation

The site has been evaluated using a combination of automated and manual checks, including:

  • axe DevTools (browser extension)
  • Lighthouse (Chrome DevTools – Accessibility audit)
  • Manual keyboard testing (Tab / Shift+Tab, focus order, form submission states)
  • WAVE (...)

Known limitations

As of the current state, the website does not have any known content that is not conformant with WCAG 2.2 Level AA.

Feedback and contact

We strive to constantly improve our site. If you encounter any accessibility barriers that hinder you from fully using our site, please contact us and describe the issue and the page/state where it occurred:

accessibility@softwarelizard.example

We will get back to you as soon as possible and work on resolving the issue.

Creation of this accessibility statement

The assessment of the website's accessibility compliance was carried out in the form of a self-test as well as before mentioned tools according to WCAG 2.2 at conformance level AA in December 2025.
